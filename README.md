# Introduccion-Flexbox-CSS
Repositorio de la lección 4 del módulo 2 del curso Frontend de DEVF para aplicar lo aprendido sobre Flexbox. 

## Descripción
Este repositorio contiene una página HTML cuyo tema principal es sobre una serie animada llamada **"ARCANE"**. Se presenta la descripción y sinopsis de la serie, personajes relevantes y un formulario para unirse a la comunidad. También contiene una hoja de estilos para hacer la página web mas llamativa y que se vea mejor organizada.

## Características
En el HTML se utilizaron etiquetas como `<main>`, `<section>`, `<article>` `<nav>`, `<h1>`, `<h2>`, `<h3>`, `<ol>`, `<footer>`, entre otras para darle una mejor estructura al contenido de la página web. En el archivo CSS, se utilizaron diferentes tipos de selectores (elementos, clase y ID) y diferentes atributos para darle un estilo único al sitio web, cambiando la tipografía, el color de fondo, el tamaño de la fuente, el tamaño de las imagenes agregadas, etc. También se utilizó la herencia poniendo atributos como font-family, text-align y margin en el body para aplicar estos atributos a todo el cuerpo de la página, utilizando el valor de inherit en margin para forzar la herencia.
En este repositorio se crearon tarjetas flexibles para la información de los personajes utilizando la técnica de **Flexbox**, haciendo que la información se vea mucho mas llamativa e interesante para el usuario. Se utilizan propiedades como flex-direction, align-items y gap para acomodar los elementos de las tarjetas y asignar espacio entre ellos.
